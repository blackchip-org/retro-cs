## Development Notes

### Memory Map Points of Interest

| Address | Description
|-|-|
| $00d7(7) | 0 = 40 column mode, 1 = 80 column mode
| $d500    | MMU configuration register
| $d505(7) | 40/80 key, 0 = 80 (down), 1 = 40 (up)
| $d600    | VDC address/status register
| $d601    | VDC data register
